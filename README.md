# 🌱 EcoAware AI App

EcoAware AI is a domain-specific Generative AI mobile application designed to promote environmental awareness using intelligent features like AI chatbot, waste detection, and campaign generation.

---
## 📱 App Features

### 🤖 AI Chatbot
- Ask environmental questions
- Get AI-powered answers using OpenAI / Gemini

### 🎨 Poster Generator
- Generate eco-awareness posters using AI prompts

### 📍 Eco Location
- Find environmental data based on location

### 🌿 Environment Tips
- Daily tips to promote eco-friendly habits

### 🎤 Voice Assistant
- Speech-to-Text input
- AI Text-to-Speech responses

### 💬 Multi Chat System
- Create multiple chats
- Switch between chats
- Delete previous chats

---
## 🧠 Technology Stack

- **Frontend:** Flutter (Dart)
- **Backend:** API Integration
- **AI APIs:** OpenAI / Google Gemini
- **Database:** Firebase Firestore
- **Cloud:** Firebase Services
- **Voice:** Speech-to-Text & Text-to-Speech

---

## 📊 System Flow

User Input → Mobile App → API Request → AI Model → Response → Display to User
